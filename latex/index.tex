Task Description\+:

Create a universal trace library for application event logging to different channels. The library should support logging to at least one of the following channels\+: TCP, serial, console, file, etc. Additionally, it should be designed in a non-\/blocking manner to ensure it does not impede the normal operation of the applications utilizing it.

Requirements\+:


\begin{DoxyEnumerate}
\item Implement a C++ library that enables application event logging.
\item The library should support logging to at least one of the following channels\+: TCP, serial, console, file, etc. You may choose the channel that best suits your expertise or demonstrate versatility by implementing logging to multiple channels.
\item Ensure the library is non-\/blocking to prevent it from interfering with the normal execution of applications.
\item Provide clear documentation on how to use the library and its different features.
\item Optionally, you may include unit tests to ensure the reliability and robustness of your implementation.
